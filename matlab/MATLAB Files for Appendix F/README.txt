INSTRUCTIONS
The following software is required in order to use the files 
in the Appendix F for Chapter 2 Supplement folder: MATLAB 
Version 9, MATLAB's Control System Toolbox Version 10, and MATLAB's 
Symbolic Math Toolbox Version 6. Follow the 
installation instructions included with MATLAB.

Be sure the folder "MATLAB files in Appendix F for Chapter 2" is 
added to the search path in Set Path in the Environment section 
under the Home tab. 


To see the computer responses after installing the M-files, 
run each problem by typing the M-file name, such as ch2sp1, 
after the prompt (>>) in the Command window. To view all or 
part of the M-file in the Command window, enter 
"type <file name>" or "help <file name>", respectively, after 
the prompt.
